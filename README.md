# Mathematical Set Visualisation

Fun Visualization of different mathematical sets

Features:
 * Mandelbrot Set
 * Julia Set
 * Fatou Set
 * Zoom In functionality
 * Save image
 * Custom resolution

You will need:
* python 3 
* PyQt 5

Run the **src/mqt.py** file
