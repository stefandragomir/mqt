# Mathematical Set Visualisation

Fun Visualization of different mathematical sets

Features:
 * View Mandelbrot Set
 * View Julia Set (not done yet)
 * View Fatou Set (not done yet)
 * Zoom In
 * Save Image
 * Custom resolution

How to run:
 * run the **src/mqt.py** file

You will need:
* python 3 
* PyQt 5
